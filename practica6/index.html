Práctica 6: Sensores digitales, analógicos e inteligentes

Ximena Cisneros Rosete
Ingeniería Biomédica
Fecha: Octubre 5, 2023
Arduino Electrónica Attiny85 Sensores

- Resumen

En esta práctica lo que se realizó fue una práctica donde se ponian a prueba diferentes sensores, ya sean digitales, analógicos e inteligentes.

Es importante mencionar se siguió utilizando el Attiny85 y Arduino para programar. Con la diferencia que en este caso nosotros escribimos el código pertinente para hiciera lo que tenía que hacer la pantalla OLED.

- Introducción

Es un dispositivo diseñado para captar un estímulo de su entorno y traducir esa información que recibe. Esa información recibida es normalmente convertida a un impulso eléctrico que posteriormente es procesado por una serie de circuitos que generan una acción predeterminada en un aparato, sistema o máquina. Es un artefacto que en algunas aplicaciones transforma una clase de información en otra que se quiere medir o controlar.

El Microcontrolador ATTINY85 es usado en proyectos wearable y por supuesto en proyectos portables con los cuales podrás implementar los protocolos de comunicación más populares como SPI o I2C, incluso programarlos para PWM. Es practico para montajes pequeños en donde un Arduino queda muy sobrado (tanto en espacio físico y memoria).

- Materiales

Para esta práctica solamente se utilizó un Arduino UNO, cables, Attiny85, 2 sensores digitales, 2 sensores analógicos y 2 inteligentes.

- Desarrollo

Lo primero que se realizó fue programar el Arduino UNO para que se ocupara como un programador, de ahí se configuró para que el Attiny pudiera estar de alguna forma conectado con el Arduino, y de ahí se programo todo para que pudiera hacer lo que la actividad prendía.

Se tenía que programar los diferentes sensores para que se mostrara lo que hacian o cómo funcionaban.

Simulación

A continuación se muestran las simulaciones realizadas:

Sensores digitales:

- Simulación del primer sensor digital, que cuando se aprieta el botón prende un LED

- Simulación del segundo sensor digital, que es un dipswitch, que al subir un switch se prende un LED y viceversa.

Sensores analógicos:

- Simulación del primer sensor analógico, que es un potenciometro, que al girar la perilla a su totalidad, se prende un LED

- Simulación del segundo sensor analógico, que es un sensor de temperatura, que a 25° o mas se enciende el LED

Sensores inteligentes:

- Simulación del primer sensor inteligente, que es un sensor ultrasónico, que cuando se acerca algo a 5 cm o menos, se enciende un LED

- Simulación del segundo sensor analógico, que es un sensor de humedad, que al revasar 25° de humedad se prende un LED, en caso contrario no

Se puede concluir que esta práctica ayudó demasiado en el poder entender el funcionamiento del Attiny85, pero más que eso logré comprender como funcionaba la pantalla OLED y configurarlo con el botón.
